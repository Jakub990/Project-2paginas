Wat doet een software developer Nou eigenlijk?

Een software developer maakt websites,games en nog veel meer!
De mogelijkheden zijn (bijna) oneindig.
Je kan apps maken voor op je telefoon of tablet,
Je kan ook systemen maken voor bedrijven,
Je kan ook databases maken voor bedrijven.
